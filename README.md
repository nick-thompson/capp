# capp

> Uber-simple CouchApp deployment in Node.js.

## Installation

npm install -g capp

## Usage

```
Usage: capp [command] [arg1, [arg2, [...]]]

Commands:
  init                Scaffold a quick CouchApp in the current working directory.
  push [path] [uri]   Push a CouchApp to a database.

```
